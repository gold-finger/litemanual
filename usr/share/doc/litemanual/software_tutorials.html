KeePassX


KeePassX is a password manager program used for securely creating and storing passwords. It includes a password generator that will create strong passwords for you. When using KeePassX, you only need to remember one password -- the one you use for opening the program! Once your KeePassX data file is opened, transferring a password from the program to its entry form is done quickly and easily with a few mouse clicks.

To install, go to Menu -> System -> Install Additional Programs, type the number corresponding to KeePassX from the list shown in the pop-up window and enter your password when prompted for it.

After it's installed you can start it from Menu -> Accessories -> KeePassX. On your first startup you will need to create a new database for your passwords -- click the interface button for that in the upper-left corner.


You will then be prompted to make a master password for the database that will contain your passwords. This is the only password you will need to remember. Do your best to make one that you not only can remember, but is strong also. Make it reasonably long with a combination of letters, numbers and other characters if possible.


After entering your password you will see that the database is now populated with two groups shown on the left: Internet and eMail. Groups are just ways to organize your passwords according to their use. (You can edit, delete and create new groups by clicking 'Groups' along the top of the window.)


To create a password entry, click the 'Add New Entry' button along the top.


Select which group the password should fall under, fill-in your information, then enter your password (if you already have one) or click on the 'Gen' button to bring up the Password Generator to make one for you.


In the Password Generator you can specify the types of characters to include, the password length, etc., along with the option to enable entropy collection to further strengthen the password.


If you enable entropy collection move your mouse around randomly when the collection window pops up. The mouse data will be incorporated by the algorithm to produce the random password.


When it's done, hit 'OK' and it will fill-in the password it produced for you. If you would like to see it, click the eye symbol next to the password box.


Click 'OK' when you are satisfied with everything and your new password entry will now appear in the list for the group you put it under. When you are done making your entries (or changes to existing ones), make sure save everything by clicking to click the Save Database button along the top of the window.


A dialog box will pop-up for you to specify a name for the file and where to save it. It will automatically put the '.kdb' (keepassx data base) extension on the file name and it will default to saving it in your home folder. (You can designate a different location to save to if you'd like.) So the example below will save a file called 'my-keepassx-data.kdb' under /home/bill.


VERY IMPORTANT!
Be sure to make at least one extra copy of your '.kdb' file to an external source, like a USB backup stick. If your hard drive fails, or something else goes wrong that prevents you from retrieving your files you will lose all of your passwords and access to the accounts you set them for. That will not be a pleasant experience if you don't have a backup copy made -- so do that immediately and update it every time you make changes to the database.


Using KeePassX is simple!

Start KeePassX and highlight the site you want to log into. Navigate to the website, then click the person icon along the top of the KeePassX window to copy your username to the clipboard temporarily. Right-click in the website entry field for it and paste. Click the key icon (next to the person icon) to copy the password to the clipboard and paste it into the website password field. That's all there is to it! (Note: the information is automatically deleted from the clipboard after 20 seconds. You can change that and other settings by going to Extras -> Settings -> Security.)

Plank Dock


Plank is a lightweight docking application for program launchers and works well with the Xfce desktop used in Linux Lite. Conveniently add launchers for all of your favorite programs on the launcher for quick, easy access. The Plank dock can be positioned along any edge of the screen. You can start the dock from the main menu or have it start automatically upon boot-up of the computer.

Plank is one of the applications you can add easily from the Install Additional Software section in the Menu. The Plank installer in Linux Lite will also download and install many different themes for the dock and put a dark plank-themer-ondock icon on the dock, which you can click to list and choose from the different themes.


The dock themes will not display properly unless compositing is enabled; so do the following to make sure you have it enabled.

Go to Menu -> Settings -> Settings Manager, then choose Window Manager Tweaks.


Click the Compositor tab and click "Enable display compositing".


Now you're ready to install Plank. Go to Menu -> System -> Install Additional Software and choose Plank Dock from the applications listed in the pop-up.


When installation is complete you will find Plank listed under Menu -> Accessories.


POSITIONING THE DOCK

Plank's default position is centered along the bottom of your screen display. If your panel is also along the bottom it will block part of the dock. You will need to move either the dock or the panel to a different position. Either one of them can be moved to the top, bottom, or either side of the screen.


  • To Move the Panel:

    • Right-click an empty area on the panel
    • Choose: Panel -> Panel Preferences -> Display (tab)
    • Uncheck the "Lock Panel" box
    • With mouse, grab either end of the panel to move it
    • Lock the panel again after move to new position
    • Note: change 'Mode' to 'Deskbar' or 'Vertical' if you want the panel on one of the sides of the display.


  • To Move the Plank Dock:

    • Open your file manager, click View -> Show Hidden Files
    • Navigate to .config/plank/dock1 and open the settings file (with leafpad text editor).
    • Or, from Terminal enter this command: leafpad ~/.config/plank/dock1/settings
    • In opened settings file (see pic below under 'All Other Plank Dock Settings' heading):
      • Change value in line #18 according to the description on previous line #17
      • Save the file
    • If setting did not take immediate effect, right-click the Plank icon on the dock and select 'Quit' to close the dock.
    • Re-start the dock from Menu -> Accessories -> Plank



After the move operation you will be able to see the dock properly.


ADDING PROGRAM LAUNCHERS TO THE DOCK

When you open a program from the Menu it will then appear on the dock. When you close the application its icon will then disappear from the dock. For programs you use often, you'll want to keep them on the dock for easy access so you don't need to navigate the Menu. There are two basic ways to accomplish that:

  1. Open a program through the Menu, right-click the icon that appears on the dock and choose 'Keep in Dock'.
  2. Open the Menu, find program to add, then drag it and drop it on the dock.


REMOVING PROGRAM LAUNCHERS FROM THE DOCK

Two ways to remove items from the dock are:

  1. Right-click the launcher icon and un-select 'Keep on Dock'.
  2. Simply grab, drag and drop the icon somewhere off of the dock.


ARRANGING LAUNCHER POSITIONS ON THE DOCK

You can change the order of items on the dock by grabbing them with your mouse and moving them to the right or left and dropping them into place. The dark plank-themer-ondock icon is the only one you can not grab. To change its position on the dock, move the other icons to its left or right until it ends up where you'd like it to reside.

LAUNCHING MORE THAN ONE INSTANCE OF A PROGRAM

When you click the launcher of a program that is already running, it will bring that running window into focus on the screen (bring it to the foreground). If instead you would like to launch another instance of the program then:

  • Middle mouse click on the launcher icon, or
  • Use 'Ctrl' key while left-clicking the icon with your mouse.


SETUP PLANK TO START AUTOMATICALLY ON BOOT

Go to Menu -> Settings -> Session and Startup -> Application Autostart (tab)

  • Hit '+' button to add new startup entry
  • Name = Plank Dock
  • Description = Start Plank dock on startup
  • Command = plank
  • Click 'OK' when done

Close the Session and Startup window when done. Your next reboot will start Plank automatically.


ALL OTHER PLANK DOCK SETTINGS

All other settings can be adjusted through the dock's settings file. Use your file manager or the terminal to open file located in the ~/.config/plank/dock1 directory. (Remember to set file manager to 'View Hidden Files'.) You can edit various lines in the file to make your changes. If changes do not take effect immediately after saving the changed file, quit and restart Plank. Please refer to the following picture while reading about the various settings that can be changed.

(Note: we hope to have a working GUI interface for these settings available within a couple of months, time permitting.)


Changes that you can make by editing the settings file are:

  • (Lines #5-6) Show only windows of the current workspace on the dock.
    • False = icons show on dock for programs opened on any workspace
    • True = only icons for the current workspace show on the dock

  • (Lines #7-8) Change the size of the launcher icons on the dock

  • (Lines #9-12) Control hiding behavior and time delay for hiding the dock

  • (Lines #13-14) Set which monitor to show the dock on if you have more than one.

  • (Lines #15-16) DO NOT MODIFY

  • (Lines #17-18) Specify which edge of the screen to position dock on.

  • (Lines #19-20) Offset the dock from center position
    • 20 would offset the dock to the right of center by 20 percent
    • -20 would offset the dock to the left of center by 20 percent

  • (Lines #21-22) Name of dock theme to use. This is handled automatically by using the themer icon on the dock. You can change themes manually here if you need to.

  • (Lines #23-24) Change the dock alignment on the screen edge it is on. For example, if you have the dock along the bottom edge, you can center it, align left, or align it right. You can also choose 'panel-mode' to make the dock appear like a panel instead of a dock.

  • (Lines 25-26) If 'panel-mode' was chosen, you can align the launcher icons on the panel

  • (Lines 27-28) Enable or disable the ability to drag and drop items from the panel for deletion. If you find that you have problems accidentally dragging things off the dock and deleting them, change this setting.

  • (Lines 29-30) Pressure-based revealing of the dock if that is supported.
